# Metric Contract

## Default Units

- pipeline, forecast, won, lost, slipped, and coverage metrics: `ARR (€ converted)`
- renewals and churn-related renewal amount metrics: `ACV (€ converted)`

## Horizon Rules

Always label the horizon explicitly:

- `All Open`
- `FY26`
- `Q2 2026`
- `Q1 2026 actuals`

Never collapse these into a single unlabeled pipeline number.

## Pipeline Rules

- exclude `Omitted` from active headline pipeline unless the omitted amount is explicitly called out
- if omitted is shown, label it separately
- approval-rate commentary applies to stage 3+ land opportunities only when the source tab says so

## Commercial Approval Rules

- include approved deals and missing-approval candidates separately
- call out stage-3 land deals with no approval
- if approval rules are missing or sparse, say that directly

## Renewals Rules

- renewal amount must be `ACV (€ converted)`
- if risk tagging is sparse, say it is sparse
- do not present org-wide retention reference KPIs as director commitments

## Q1 Rules

- won and lost actuals can be used as delivered outcomes
- slipped deals can be used when director-scoped
- workbook Q1 forecast blocks may be global reference only
- do not label ambiguous Q1 forecast reference values as the director promise
