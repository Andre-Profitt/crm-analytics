# Output Schema

Return exactly these headings in markdown:

## Executive Summary

- exactly 3 bullets

## Pipeline Overview

- exactly 3 bullets
- include all-open, FY26, and Q2 framing where available

## Commercial Approval

- exactly 3 bullets
- include approved state and missing-approval candidates

## Renewals

- exactly 3 bullets
- use ACV

## Q1 Promise vs Delivered

- exactly 3 bullets
- separate delivered actuals from any ambiguous promise baseline

## Slipped Deals

- exactly 3 bullets
- use factual slippage observations only

## Coverage and Intel

- exactly 3 bullets
- include pipeline coverage, forecast mix, risk concentration, hygiene, or competitive-loss intel

## Open Questions

- at most 2 bullets
- only include unresolved issues that materially block clean executive reporting
