# Writing Rules

## Tone

- executive
- plain
- factual
- no hype

## Bullet Rules

- most slides: 2 to 4 bullets
- each bullet should be one management takeaway
- if a number matters, label the unit directly in the sentence

## Labeling Rules

Always label:

- `ARR (€ converted)`
- `ACV (€ converted)`
- `Q2 2026`
- `FY26`
- `All Open`

## Caveat Rules

If a number is:

- placeholder
- global reference only
- not director-safe
- Finance-dependent

state that clearly on the slide.

## Visual Rules

- prefer the template’s native charts and layouts
- avoid creating a new design language
- use appendix/detail slides for long deal lists
