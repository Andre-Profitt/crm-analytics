# Context Files

This folder keeps the working context small without losing the supporting research.

Use these files in order:

1. `ACTIVE_CONTEXT.md`
- what is hot right now
- which dashboard is active
- which files should be in working memory

2. `EVIDENCE_INDEX.md`
- what supporting evidence exists
- where it lives
- whether it is active, supporting, or archival

3. `templates/DASHBOARD_BRIEF_TEMPLATE.md`
- structure for future dashboard briefs and cycle resets

The machine-readable companion lives at:
- `/Users/test/crm-analytics/config/context_registry.json`

Generated research, audits, and exports remain under:
- `/Users/test/crm-analytics/docs/generated`

Those generated files are preserved context, not always hot context.
