# Publish Checklist

## Required Inputs

- workbook snapshot exists
- validated fact pack exists
- validation report exists
- PowerPoint review exists or an explicit reason exists for skipping it

## Required Slide Coverage

- executive summary
- pipeline overview with quarterly focus
- commercial approval overview
- renewals tracking
- Q1 promised vs delivered
- slipped deals and commentary gap
- coverage or execution intel

## Required Number Quality

- pipeline uses ARR with horizon labels
- renewals use ACV
- omitted is not hidden inside active headline pipeline
- any global reference blocks are qualified

## Required Narrative Quality

- one message per slide
- executive, not spreadsheet-like
- appendix carries detail if necessary

## Blocker Examples

- deck still shows stale pre-fix pipeline numbers
- PowerPoint review found number framing mismatches
- Excel-Claude brief was never validated
- missing commercial approval candidates are not named
